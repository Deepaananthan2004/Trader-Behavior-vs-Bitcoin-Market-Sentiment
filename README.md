# Trader Behavior vs Bitcoin Market Sentiment Analysis

## Overview

This project analyzes the relationship between cryptocurrency trader performance and Bitcoin Market Sentiment (Fear & Greed Index).

The objective is to determine whether market sentiment influences trading profitability, trading behavior, and trader performance.

---

## Dataset

### Historical Trader Data

Contains

- Account
- Coin
- Execution Price
- Trade Size
- Closed PnL
- Direction
- Timestamp
- Fee
- Leverage
- etc.

### Bitcoin Fear & Greed Dataset

Contains

- Date
- Classification
- Fear
- Greed
- Neutral
- Extreme Greed

## Key Insights

- Fear dominated the observed trading periods.
- Trader profitability varied significantly across market sentiment.
- Large trades generated higher volatility.
- Certain coins consistently produced higher returns.
- Buy and Sell strategies showed different profitability under different sentiment conditions.
